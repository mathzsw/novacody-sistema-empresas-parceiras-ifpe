# Sistema de Cadastro de Empresas Parceiras

Projeto desenvolvido para a disciplina Projeto e Prática 2 (PP2) do Instituto Federal de Pernambuco (IFPE) – Campus Jaboatão dos Guararapes.

## Equipe
- Matheus Augusto Rodrigues de Alencar
- Thais Myller

## Empresa
**Novacody**

## Descrição
O projeto tem como objetivo desenvolver um sistema web para cadastro e gerenciamento de empresas parceiras do IFPE, permitindo o cadastro, consulta, atualização e aprovação de empresas, além da administração do sistema por usuários autorizados.

## Funcionalidades

### CRUD de Empresas
**Desenvolvido por:** Matheus Augusto Rodrigues de Alencar

- Cadastro de empresas parceiras
- Consulta de empresas cadastradas
- Edição de empresas
- Exclusão de empresas
- Aprovação de empresas pelo painel administrativo
- Busca por nome, CNPJ, e-mail e contato
- Filtro por tipo de parceria

### CRUD de Administradores
**Desenvolvido por:** Thais Myller

- Cadastro de administradores
- Consulta de administradores
- Edição de administradores
- Exclusão de administradores
