# Sistema de Cadastro de Empresas Parceiras

Projeto desenvolvido para a disciplina Projeto e Prática 2 (PP2) do IFPE Campus Jaboatão dos Guararapes.

## Equipe
- Matheus Augusto Rodrigues de Alencar
- Thais Myller

## Empresa
Novacody

## Descrição
O projeto tem como objetivo desenvolver um sistema web para cadastro e gerenciamento de empresas parceiras do IFPE, permitindo o cadastro, consulta, atualização e divulgação de oportunidades para os alunos.

## Funcionalidades
- Cadastro de empresas parceiras
- Consulta de empresas cadastradas
- Gerenciamento de usuários
- Divulgação de oportunidades
- Aprovação de empresas

## Tecnologias
- HTML
- CSS
- JavaScript
- Node.js
- Express
- MySQL
